# DymPaint
Paint application for daily usage based on java swing
## Features
* Brush, Rectangle, Circle and Line draw modes
* Adjustable stroke
* Ready to use 8 colors and 1 color choser
* Fill tool for background and shapes
* Color picker
* Eraser
* Undo and redo
